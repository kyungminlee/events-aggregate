# Bay Area Kids & Family Events Aggregator

A static site that aggregates kids and family events from Bay Area city websites and public libraries, automatically updated daily via GitHub Actions.

## Live Site

Deployed to GitHub Pages — enable it under **Settings → Pages → Source: GitHub Actions**.

## Sources

### City Websites
| City | URL | Platform |
|------|-----|----------|
| San Jose | sanjoseca.gov | CivicPlus |
| Mountain View | mountainview.gov | CivicPlus |
| Sunnyvale | sunnyvale.ca.gov | govAccess |
| Palo Alto | paloalto.gov | OpenCities |
| Menlo Park | menlopark.gov | OpenCities |

### Libraries
| Library | URL | Platform |
|---------|-----|----------|
| Santa Clara County Library (SCCL) | sccl.bibliocommons.com | BiblioCommons API |
| San Jose Public Library | sjpl.org | HTML scrape |

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Run scrapers (fetches live data → site/data/events.json)
python run_scrapers.py

# Or fetch kids-only events for the next 90 days
python run_scrapers.py --kids-only --days 90

# Serve the site locally
python -m http.server 8000 --directory site
# Open http://localhost:8000
```

## Tuning Scrapers

Each scraper tries its best with the site's public HTML/API. If a scraper returns zero results, inspect the target site's actual HTML and update the CSS selectors in the relevant file. CivicPlus sites may require updating `calendar_ids` after inspecting network requests in browser DevTools.

## Kids/Family Detection

Events are tagged `is_kids_event: true` if their title, description, or categories contain any of these keywords:

> kid, kids, child, children, youth, family, families, teen, teens, toddler, baby, infant, preschool, storytime, story time, puppet, craft, summer reading, after school, elementary, homework help, lego, young adult, jr., kindergarten, playdate
